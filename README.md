# 🌤 Aesthetic Weather App (Python + Tkinter)

A modern and visually appealing **Weather App** built using **Python (Tkinter GUI)**.  
This app fetches **live weather data** based on user-selected location and displays it in a clean, aesthetic interface inspired by mobile weather apps.

## 🚀 Features

- 🌍 Search any city (live weather)
- 🌡 Real-time temperature display
- ⛅ Weather condition with icons
- 💧 Humidity, 🌬 wind, 🌫 visibility, 📊 pressure
- ☀️ UV index & 🌼 Pollen level
- 🌅 Sunrise & 🌇 Sunset time
- 🧠 Smart lifestyle tips
- 🎨 Aesthetic UI with background image
- 📱 Mobile-style layout

---

## 🛠 Tech Stack

- **Python 3**
- **Tkinter (GUI)**
- **Requests (API calls)**
- **Pillow (image handling)**
- **Open-Meteo API**

## 🎮 How to Use
Enter city name (e.g. Noida, Delhi, Mumbai)
Click Search
View:
Temperature 🌡
Weather condition ⛅
Pollen 🌼
Extra stats (UV, wind, pressure)
Lifestyle tips 🧠
⚠️ Notes
Internet connection required 🌐
Pollen data may show N/A in some regions
